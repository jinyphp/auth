# 회원 서비스 이용 가이드

본 문서는 사용자가 사이트의 회원 서비스를 이용하는 방법에 대해 설명합니다.

---

## 1. 회원가입

사이트의 모든 서비스를 이용하기 위해서는 회원가입이 필요합니다.

### 1.1 가입 절차

1.  **회원가입 페이지 이동**: 상단 메뉴의 '로그인' 버튼을 클릭한 후, 로그인 폼 하단의 '회원가입' 링크를 클릭하거나, `/signup/terms` 주소로 직접 접속합니다.
2.  **약관 동의**:
    *   서비스 이용약관 및 개인정보 처리방침 등 **필수 약관**에 반드시 동의해야 합니다.
    *   마케팅 정보 수신 동의 등 **선택 약관**은 동의하지 않아도 가입이 가능합니다.
    *   '전체 동의'를 체크하면 모든 약관에 한 번에 동의할 수 있습니다.
    *   동의 후 '다음 단계' 버튼을 클릭합니다.
3.  **회원 정보 입력**:
    *   **이름**: 사용자 이름을 입력합니다.
    *   **이메일**: 로그인 아이디로 사용할 이메일 주소를 입력합니다. (중복된 이메일은 사용할 수 없습니다.)
    *   **비밀번호**:
        *   최소 8자 이상이어야 합니다.
        *   영문 대소문자, 숫자, 특수문자를 포함해야 할 수 있습니다(설정에 따라 다름).
        *   비밀번호 확인란에 동일한 비밀번호를 한 번 더 입력합니다.
    *   **기타 정보**: 국가, 언어 등 추가 정보를 입력합니다 (선택 사항).
4.  **가입 완료**:
    *   '회원가입' 버튼을 클릭합니다.
    *   입력한 정보에 오류가 없다면 회원가입이 완료되며, 가입 성공 페이지로 이동합니다.
    *   _참고: 이메일 인증이 필요한 경우, 가입 후 인증 메일이 발송될 수 있습니다._

---

## 2. 로그인

가입한 계정으로 로그인하여 서비스를 이용할 수 있습니다.

1.  **로그인 페이지 이동**: `/login` 주소로 접속합니다.
2.  **정보 입력**:
    *   가입 시 입력한 **이메일**과 **비밀번호**를 입력합니다.
3.  **로그인 시도**:
    *   '로그인' 버튼을 클릭합니다.
    *   정보가 일치하면 로그인되어 메인 페이지나 이전 페이지로 이동합니다.
    *   5회 이상 로그인 실패 시, 보안을 위해 일시적으로 로그인이 제한될 수 있습니다.

---

## 3. 비밀번호 찾기 (재설정)

비밀번호를 잊어버린 경우, 이메일 인증을 통해 비밀번호를 재설정할 수 있습니다.

1.  **비밀번호 찾기 페이지 이동**:
    *   로그인 페이지 하단의 '비밀번호를 잊으셨나요?' 링크를 클릭하거나, `/signin/password/reset` 주소로 접속합니다.
2.  **이메일 인증 요청**:
    *   가입 시 사용한 **이메일 주소**를 입력합니다.
    *   '비밀번호 재설정 링크 발송' 버튼을 클릭합니다.
    *   가입된 이메일인 경우, 해당 메일함으로 재설정 링크가 포함된 이메일이 발송됩니다.
3.  **비밀번호 변경**:
    *   수신된 이메일의 '비밀번호 재설정' 링크를 클릭합니다.
    *   새로운 비밀번호를 입력하는 페이지(`/signin/password/reset/{token}`)가 열립니다.
    *   사용할 **새 비밀번호**를 입력하고 확인란에 다시 입력합니다.
    *   '비밀번호 재설정' 버튼을 클릭하면 비밀번호 변경이 완료됩니다.
4.  **로그인**: 변경된 비밀번호로 다시 로그인합니다.

---

## 4. 자주 묻는 질문 (FAQ)

**Q. 약관 동의 페이지가 계속 나와요.**
A. 회원가입 시 필수 약관에 동의하지 않으면 가입을 진행할 수 없습니다. 모든 필수 항목에 체크했는지 확인해주세요. 로그인 페이지로 이동했다가 다시 회원가입을 시도하면 동의 내역이 초기화됩니다.

**Q. "이미 사용 중인 이메일입니다" 오류가 떠요.**
A. 해당 이메일로 이미 가입된 계정이 있습니다. 비밀번호를 잊으셨다면 '비밀번호 찾기'를 이용해주세요.

**Q. 비밀번호 재설정 메일이 오지 않아요.**
A. 스팸 메일함을 확인해보세요. 그래도 없다면 입력한 이메일 주소가 정확한지, 가입된 이메일이 맞는지 확인해주세요.
